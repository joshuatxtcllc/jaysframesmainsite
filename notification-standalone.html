About the Notification System

The Jay's Frames notification system is designed to provide real-time updates to users about their orders, new product arrivals, special promotions, and system alerts.

Features

  • Real-time notifications
  • Multiple notification types (info, success, warning, error)
  • Notification badge counter
  • Toast notifications
  • Notification history
  • Mark as read functionality

This demo showcases the client-side functionality of the notification system without requiring a server connection.
