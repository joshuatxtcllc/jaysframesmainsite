About the Notification System

The Jay's Frames notification system provides real-time notifications across all applications in the Jay's Frames ecosystem. It features:

  • Real-time WebSocket communication
  • Multiple notification types (info, success, warning, error)
  • Actionable notifications with links
  • Badge counters for unread notifications
  • Toast notifications for immediate visibility
  • Embeddable client for third-party applications

This demo simulates the client-side functionality of the notification system.
